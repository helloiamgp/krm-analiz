KRM ANALİZ ARACI - WINDOWS EXE KULLANIM KILAVUZU
================================================================================

📦 EXE OLUŞTURMA

1. ADIM: GEREKSİNİMLER
   - Python 3.8 veya üstü kurulu olmalı
   - pip kurulu olmalı

   Kontrol için terminal'de:
   > python --version
   > pip --version

2. ADIM: BAĞIMLILIKLARI YÜKLE
   Terminal'de şu komutu çalıştırın:
   > pip install -r requirements.txt

3. ADIM: EXE OLUŞTUR
   build_exe.bat dosyasını çift tıklayın VEYA terminal'de:
   > build_exe.bat

4. ADIM: ÇIKTI KONTROL
   Başarılı olursa:
   - dist\KRM-Analiz.exe dosyası oluşur
   - Bu dosya bağımsız çalışır (Python gerektirmez)

================================================================================

💻 EXE KULLANIMI

1. HAZIRLIK
   KRM-Analiz.exe dosyasını istediğiniz dizine kopyalayın.
   Örnek:
   C:\KRM\
   ├── KRM-Analiz.exe
   └── [PDF dosyalarınız buraya]

2. ÇALIŞTIRMA
   KRM-Analiz.exe dosyasını çift tıklayın VEYA terminal'de:
   > KRM-Analiz.exe

3. SONUÇLAR
   Program otomatik olarak:
   - Dizindeki tüm PDF dosyalarını tarar
   - Analiz eder
   - output\ dizinine raporları kaydeder

   Örnek Çıktı:
   C:\KRM\
   ├── KRM-Analiz.exe
   ├── Banka-KRM.pdf
   └── output\
       └── Banka-KRM.pdf (analiz raporu)

❓ SIKÇA SORULAN SORULAR

S: EXE dosyası neden büyük?
C: PyInstaller tüm Python runtime ve kütüphaneleri içerir (~50-100 MB).

S: Antivirüs programı uyarı veriyor?
C: PyInstaller ile oluşturulan dosyalar bazen false-positive tetikler.
   Güvenli kaynaklıysa whitelist'e ekleyin.

S: macOS/Linux için exe oluşturabilir miyim?
C: Evet! build_exe.sh scriptini kullanın.
   > chmod +x build_exe.sh
   > ./build_exe.sh

S: Birden fazla PDF'i nasıl analiz ederim?
C: Tüm PDF'leri KRM-Analiz.exe ile aynı dizine koyun.
   Program otomatik olarak hepsini işler.

S: Tek bir PDF için nasıl kullanırım?
C: Terminal'de:
   > KRM-Analiz.exe "dosya-adi.pdf"
